Hoja de Trucos: Arreglos y Objetos en JavaScript

Declaración de Arreglo

Los arreglos en JavaScript son ordenados, lo que significa que los elementos se almacenan en una secuencia específica.

const fruits = ["apple", "banana", "cherry"];
console.log(fruits);

Indexación de Arreglos

Los arreglos se indexan desde cero, el primer elemento está en el índice 0.

const fruits = ["apple", "banana", "cherry"];
const firstFruit = fruits[0]; // "apple"
const secondFruit = fruits[1]; // "banana"
console.log(firstFruit, secondFruit);

Longitud del Array

La propiedad length se utiliza para determinar el número de elementos en un array.

const fruits = ["apple", "banana", "cherry"];
console.log(fruits.length); // 3

Mutabilidad de Arreglos

Los arreglos son mutables, puedes cambiar, agregar o eliminar elementos después de crearlos.

const fruits = ["apple", "banana", "cherry"];
fruits[2] = "strawberry"; // Modificar un elemento
fruits[3] = "Kiwi";        // Agregar un elemento
console.log(fruits);

Método push

Agrega uno o más elementos al final de un arreglo.

const fruits = ["apple", "banana"];
fruits.push("orange", "strawberry");
console.log(fruits);

Método pop

Elimina el último elemento de un array y lo devuelve.

const fruits = ["apple", "banana", "orange"];
const removedFruit = fruits.pop();
console.log('Fruits are', fruits);
console.log('Removed fruits are', removedFruit);

Método splice

Cambia el contenido de un array eliminando, reemplazando o añadiendo elementos en una posición específica.

const fruits = ["apple", "banana", "cherry"];
fruits.splice(1, 1, "grape"); // Reemplaza el segundo elemento con "grape"
console.log(fruits);

Método concat

Combina arreglos en secuencia creando un nuevo arreglo.

const fruits = ["apple", "banana"];
const additionalFruits = ["orange", "strawberry"];
const combinedFruits = fruits.concat(additionalFruits);
console.log('combinedFruits are', combinedFruits);

Método slice

Devuelve una copia superficial de una porción de un array.

const fruits = ["apple", "banana", "cherry", "orange"];
const slicedFruits = fruits.slice(1, 3); // Desde índice 1 hasta 2
console.log('slicedFruits are', slicedFruits);

Método indexOf

Encuentra el índice de la primera ocurrencia de un elemento en un arreglo.

const fruits = ["apple", "banana", "cherry", "banana"];
const index = fruits.indexOf("banana"); // 1
console.log('Index of banana is', index);

Método reverse

Invierte el orden de los elementos en un arreglo.

const fruits = ["apple", "banana", "cherry"];
fruits.reverse();
console.log(fruits);

Método sort

Ordena los elementos de un array en su lugar.

const numbers = [4, 2, 8, 6, 1, 10];
numbers.sort(); // Ordena como strings
numbers.sort((a, b) => a - b); // Ordena como números
console.log(numbers);

Iteración de arreglos

Se puede usar un bucle for para recorrer los elementos de un arreglo.

const fruits = ['apple', 'banana', 'cherry', 'date'];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Método forEach

Itera un arreglo y aplica una función a cada elemento.

function sendWelcomeEmail(email) {
    console.log(`Welcome email sent to ${email}`);
}
const users = [
    { name: 'Alice', email: 'alice@example.com' },
    { name: 'Bob', email: 'bob@example.com' },
    { name: 'Charlie', email: 'charlie@example.com' },
];
users.forEach((user) => {
    sendWelcomeEmail(user.email);
});

Método map

Crea un nuevo array aplicando una función a cada elemento.

const products = [
    { name: 'Laptop', price: 1000 },
    { name: 'Smartphone', price: 500 },
    { name: 'Tablet', price: 300 },
];
products.map((product) => {
    console.log(`The price of ${product.name} is $${product.price}`);
});

Método filter

Crea un nuevo array que contiene elementos que cumplen una condición.

const products = [
    { name: 'Laptop', price: 1000 },
    { name: 'Smartphone', price: 500 },
    { name: 'Tablet', price: 300 },
    { name: 'Monitor', price: 250 },
    { name: 'Keyboard', price: 50 },
];
const filteredProducts = products.filter(product => product.price >= 100 && product.price <= 500);
filteredProducts.forEach(product => console.log(`${product.name} is $${product.price}`));

Método reduce

Reduce un array a un solo valor aplicando una función a cada elemento.

const orderPrices = [50, 30, 25, 40, 15];
const totalOrderValue = orderPrices.reduce((total, price) => total + price, 0);
console.log('The total value of order is', totalOrderValue);

Método find

Devuelve el primer elemento que satisface una condición específica.

const employees = [
    { name: 'Alice', age: 35 },
    { name: 'Bob', age: 32 },
    { name: 'Charlie', age: 38 }
];
const employee = employees.find(employee => employee.name === "Charlie");
console.log(employee.age);
