Assing an initial value

name = "John"; // initializing a variables

Declaration : Declaration keywords
  • VAR: Function-level Scope - a variable declared with VAR is available throughout the entire function
  • LET: Block-level Scoping - is only available within the block enclosed by curly brackes
  • CONST: Block-level Scoping - you can't reassing it a diferent value

LET can be reassigned but not declared wihtin the same block

Primitive data type
  • String:Text example: let name = "Alice";
  • Number: Integers and floating points example: [let age = 30;] or [let price = 9.99;]
  • Boolean: true or false values
  • Undefined: Declared but not assigned a value examples: Let x;
  • Null: Empty value example: Let emptyValue = null;
