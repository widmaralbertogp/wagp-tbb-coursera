Hoja de Trucos: Objetos en JavaScript

Clases

Las clases son una forma de crear planos o plantillas para objetos. Definen la estructura y el comportamiento de los objetos de esa clase.

class Person {
  constructor(firstName, lastName) {
    this.firstName = firstName;
    this.lastName = lastName;
  }
  getFullName() {
    return `${this.firstName} ${this.lastName}`;
  }
}
const person1 = new Person("John", "Doe");
console.log(person1.getFullName()); // Output: "John Doe"

Objetos Constructor

Los objetos son instancias de clases o pueden ser creados como objetos independientes sin una clase. Pueden tener propiedades y métodos.

class Car {
  constructor(make, model, year) {
    this.make = make;
    this.model = model;
    this.year = year;
  }
  startEngine() {
    console.log(`The ${this.make} ${this.model}'s engine is running.`);
  }
}
const myCar = new Car("Toyota", "Camry", 2022);
myCar.startEngine();

Literales de Objetos

Los literales de objetos son una forma de crear objetos ad-hoc sin definir una clase.

const person = {
  firstName: "Alice",
  lastName: "Johnson",
  getFullName: function() {
    return `${this.firstName} ${this.lastName}`;
  }
};
console.log(person.getFullName());

Constructor de Funciones

Un constructor de funciones es una función regular de JavaScript que se utiliza para crear e inicializar objetos. Es una convención nombrar a los constructores de funciones con una letra mayúscula inicial.

function Car(make, model) {
  this.make = make;
  this.model = model;
}
const car1 = new Car("Toyota", "Camry");
const car2 = new Car("Honda", "Civic");
console.log('Car1 details are', car1);
console.log('Car2 details are', car2);

Notación de Punto

La notación de punto es una forma de acceder a las propiedades de un objeto.

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 30
};
console.log(person.firstName);
console.log(person.lastName);
console.log(person.age);

Notación de Corchetes

La notación de corchetes es útil cuando los nombres de las propiedades contienen caracteres especiales o espacios.

const person = {
  "first name": "John",
  "last name": "Doe",
  age: 30
};
console.log(person["first name"]);
console.log(person["last name"]);
console.log(person["age"]);

Arrays de Objetos

Un array de objetos es una colección de múltiples objetos almacenados dentro de un único contenedor de array.

const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
console.log(students);

Acceder a un Array de Objetos

Puedes acceder a los elementos dentro de un array de objetos utilizando el índice del array y la notación de punto.

const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
console.log(students[0].name);
console.log(students[2].age);

Iterando a Través de un Arreglo de Objetos

La iteración de objetos a través de arreglos incluye bucles for y métodos de arreglo.

const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
for (let i = 0; i < students.length; i++) {
  console.log(students[i].name);
}

Agregar Objetos (push)

//Adding Elements
const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
students.push({ name: "David", age: 20 });
console.log(students);

Eliminar Objetos (pop)

//Removing Elements
const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
const removedStudent = students.pop();
console.log(students);

Filtrado y Mapeo de Arreglos de Objetos

const students = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 22 },
  { name: "Charlie", age: 28 }
];
const adults = students.filter(student => student.age >= 23);
const studentNames = students.map(student => student.name);
console.log(adults);
console.log(studentNames);

Buscando Objetos

const employees = [
